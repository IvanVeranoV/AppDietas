# AppDietas - Gestor de Menú y Compra Automatizada

## Descripción
**AppDietas** es una plataforma web integral diseñada para la planificación familiar de menús y la automatización inteligente de listas de la compra. Bajo un enfoque **"Zero-Waste"**, el sistema optimiza la logística del hogar gestionando ingredientes por unidades completas (paquetes/envases), eliminando la necesidad de micro-gestión de gramos o inventarios complejos.

El sistema permite a múltiples usuarios planificar sus comidas de forma colaborativa, consolidando automáticamente las necesidades de compra y categorizando los insumos según su naturaleza (frescos vs. no perecederos) y categoría de producto.

## Características Principales
* **Planificación Multi-usuario:** Calendario colaborativo con distinción visual por usuario.
* **Automatización de Compras:** Cálculo inteligente de cuándo comprar según la frescura del producto.
* **Consolidación de Ingredientes:** Agregación automática de paquetes necesarios para todo el grupo familiar.
* **Arquitectura "Zero-Waste":** Gestión de stock basada en unidades completas para minimizar el desperdicio.
* **Catálogo Organizado:** Clasificación por categorías de producto para optimizar el tiempo en el supermercado.

## Tecnologías Utilizadas
* **Backend:** Python con [FastAPI](https://fastapi.tiangolo.com/) para un manejo de datos eficiente y rápido.
* **Frontend:** [React.js](https://react.dev/) para una interfaz de usuario dinámica y responsiva.
* **Base de Datos:** [PostgreSQL](https://www.postgresql.org/) asegurando integridad relacional robusta.
* **Infraestructura:** Despliegue mediante contenedores [Docker](https://www.docker.com/) en Oracle Cloud.
